This is a four input multiplexer where each input is 8-bit wide. To select one of the 
four input we need 2-bit selection line. 


#SIMULATION LOG
# run 1000ns

| Sel | ValOut | expValOut |

| 0   |  0a    |  0a       |

| 1   |  0b    |  0b       |

| 2   |  0c    |  0c       |

| 3   |  0d    |  0d       |
